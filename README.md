# GoIT JS HW 11 — Image Search Gallery

Цей проект реалізує логіку пошуку та перегляду зображень. У цьому файлі описано функціонал модуля відображення інтерфейсу.

## Функціонал модуля `render-function.js`

Цей модуль відповідає за маніпуляції з DOM-деревом: створення розмітки галереї, очищення результатів та керування індикатором завантаження.

### Основні функції

1.  **`createGallery(images)`**
    *   **Призначення:** Генерує та додає розмітку зображень у DOM.
    *   **Вхідні дані:** Масив об'єктів зображень (отриманих з API).
    *   **Дії:**
        *   Створює HTML-картку для кожного зображення, що містить:
            *   Прев'ю (`webformatURL`).
            *   Посилання на велике зображення (`largeImageURL`).
            *   Блок статистики: Лайки (`likes`), Перегляди (`views`), Коментарі (`comments`), Завантаження (`downloads`).
        *   Вставляє розмітку в контейнер `.gallery`.
        *   Викликає `lightbox.refresh()` для оновлення бібліотеки SimpleLightbox (щоб нові зображення відкривалися у модальному вікні).

2.  **`clearGallery()`**
    *   **Призначення:** Очищує галерею перед новим запитом.
    *   **Дії:** Видаляє весь HTML-вміст елемента `.gallery`.

3.  **`showLoader()` та `hideLoader()`**
    *   **Призначення:** Інформування користувача про стан процесу завантаження.
    *   **Дії:** Видаляють або додають CSS-клас `is-hidden` до елемента `.loader`.

### Використані бібліотеки

*   **SimpleLightbox**: Використовується для створення модального вікна з можливістю гортання зображень. Ініціалізується на посиланнях всередині `.gallery` з налаштуваннями підписів (`captionsData: 'alt'`) та затримки (`captionDelay: 250`).
